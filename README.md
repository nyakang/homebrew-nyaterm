# homebrew-nyaterm

Homebrew tap for [NyaTerm](https://github.com/nyakang/nyaterm), a modern remote terminal workspace.

## Install

```bash
brew install nyakang/nyaterm/nyaterm
```

This will automatically tap `nyakang/homebrew-nyaterm` and install the `nyaterm` cask.

## Upgrade

```bash
brew upgrade nyaterm
```

## Uninstall

```bash
brew uninstall nyaterm
```

To remove NyaTerm user data as well:

```bash
brew uninstall --zap nyaterm
```

## Issues

For app bugs, please file issues in [nyakang/nyaterm](https://github.com/nyakang/nyaterm/issues).

For Homebrew installation problems such as an incorrect URL or checksum mismatch, open an issue in this tap repository.
